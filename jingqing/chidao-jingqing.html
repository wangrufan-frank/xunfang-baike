核心概念
采用"梯次化处置 + 模块化策应"体系，以阶段目标为导向灵活调整战术组合。5人作战小组按职能模块划分，处置前、中、后三阶段逐步升级/降级应对。
持刀类警情采用"梯次化处置 + 模块化策应"体系，以阶段目标为导向灵活调整战术组合。5人作战小组按职能模块划分，处置前、中、后三个阶段逐步升级/降级应对。

实战背景：个人极端警情频发（珠海驾车冲撞、宜兴校园砍杀），高风险区域集中在学校、体育馆、银行等人群密集场所。体系战术源于实战案例（2023年重庆银行持械伤人案）。

5人作战小组模块化分工
1号指挥（抓捕叉）→ 2/3号盾牌手（法式盾牌）→ 4号辅助手（长棍）→ 5号支援手（警戒带/围挡），各司其职形成完整作战体系。

阶段一：疏散稳控（2+2+1）
分割策应、避免激化。1/3号位安抚伤员询问报警人，2/4号位盘查持械人员疏导情绪，5号位外围封控与舆情引导。
目标：了解现场、疏散群众、明确核心对象
战术：分割策应、避免激化

· 1、3号位：伤员安抚与报警人询问
· 2、4号位：持械人员盘查与情绪疏导
· 5号位：外围封控与舆情引导
3
阶段二：作战控制（4+1）
集中优势力量，一招制敌。夹击阵型引导嫌疑人至角落，盾棍叉协同控制链：法律告知→警告无效→盾牌撞击→抓捕叉锁脚踝。
目标：消除危险、控制核心对象、防范二次风险
战术：集中优势力量，一招制敌

· 夹击阵型引导嫌疑人至角落
· 盾棍叉协同控制链：
　法律告知 → 警告无效 → 盾牌撞击 → 抓捕叉锁脚踝
4
阶段三：封控取证（3+1+1）
舆情清扫、移交程序、固定证据。1号指挥取证，2/3号上铐搜身带离，4号警戒，5号舆情封控。交接当事人及危险物品后恢复巡逻。
目标：舆情清扫、移交程序、固定周边证据

· 1号位：指挥与证据收集
· 2、3号位：上铐搜身与带离
· 4号位：警戒与舆情消除
· 5号位：舆情封控与证据固定

处置收尾：交接当事人及危险物品，恢复巡逻。

模块化策应核心原则
分阶段定目标，按目标调资源。每阶段聚焦一个主要目标，集中优势力量完成后再转入下一阶段，不追求同时完成所有目标。
以阶段目标为导向，灵活调整战术组合：
处置前期 2+2+1 → 处置中期 4+1 → 处置后期 3+1+1

核心思想：分阶段定目标，按目标调资源。不要同时追求所有目标（疏散、控制、取证），而是在每个阶段聚焦一个主要目标，集中优势力量完成后再转入下一阶段。强调多小组协同、快速补位与安全优先。
6
实战战术配合要点与处置三重点
战术配合四模式：盾叉同向/侧向站位、盾棍长棍先行/盾牌先行。处置三重点：风险衡量、安全优先、危险加一。
战术配合四模式：
· 盾与叉同向站位：盾牌保护 → 时机选择 → 快叉快拉 → 盾叉协同
· 盾与叉侧向站位：盾牌吸引 → 攻其不备 → 侧面围攻 → 灵活换位
· 盾与棍·长棍先行：控制距离 → 制造战机 → 形成合力 → 灵活堵截
· 盾与棍·盾牌先行：正面迎击 → 击打弱点 → 盾棍协同 → 乘胜追击

处置三重点：风险衡量（姿态+情绪判断）· 安全优先（地形+工具利用）· 危险加一（人数+1 + 武力+1）
