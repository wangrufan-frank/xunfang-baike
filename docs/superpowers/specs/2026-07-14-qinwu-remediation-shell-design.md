# 巡防勤务整改栏目壳恢复设计

## 目标

恢复网站与微信小程序中的“巡防勤务”模块入口，并在栏目页统一显示“内容整改中”。不得恢复此前删除的文章、摘要、搜索记录、详情路径或交叉推荐。

## 网站

- 首页恢复“巡防勤务”模块卡。
- 模块卡只显示模块名称和“内容整改中”，不显示篇数、栏目名称或旧内容热词。
- `qinwu/index.html` 保留标题和通用导航，只显示“内容整改中”。

## 微信小程序

- 首页恢复“巡防勤务”模块卡，卡片只显示“内容整改中”。
- 搜索首页模块时，仅使用“巡防勤务”和“内容整改中”两个安全词。
- `miniprogram/pages/qinwu/index/index` 不加载或搜索文章，只显示“内容整改中”。
- `miniprogram/data/qinwu.js` 继续保持空数组。
- 旧详情路径不得重新暴露内容；直接进入详情页时提示“内容整改中”并返回巡防勤务栏目页。

## 搜索与数据

- `search-index.json` 不新增巡防勤务文章记录。
- 不恢复任何已删除 HTML 文件或小程序文章数据。
- 不改动同日备份或 Git 历史。

## 验证

- 网站首页、网站栏目页、小程序首页和小程序栏目页均显示巡防勤务整改状态。
- 巡防勤务数据仍为 `module.exports = [];`。
- 24 个已删除详情文件继续不存在，禁止词、旧路径和搜索记录扫描继续通过。
- 全量自动测试通过，HTML 内链无缺失。
