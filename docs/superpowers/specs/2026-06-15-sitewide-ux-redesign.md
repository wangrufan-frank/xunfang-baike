## 一、导航系统重构 + 搜索增强

### 导航栏 JS 动态注入

**问题：** 导航 `<nav>` 代码在 60+ HTML 文件中逐页复制，变更加模块需遍历所有文件。

**方案：**
- `js/nav.js` — 包含模块配置数组和 `renderNav()` 函数
- 所有页面在 `<body>` 顶部放一个 `<div id="nav-placeholder">`，由 JS 注入导航 HTML
- 配置数组定义：模块名、路径、emoji、是否为主导航项
- `renderNav()` 根据当前 URL 自动设置 `active` 状态
- 取消"更多"下拉菜单，11 个模块全部在导航栏平铺

**模块配置结构：**
```js
const MODULES = [
  { name: '首页', path: '/index.html', emoji: '🏠' },
  { name: '警情', path: '/jingqing/', emoji: '🚨' },
  // ... 全部 11 个模块
];
```

**影响文件：**
- 新增: `js/nav.js`
- 修改: 所有 60+ HTML 文件（替换 `<nav>` 块为 `<div id="nav-placeholder">` + `<script src="../js/nav.js">`）
- 不变: CSS 样式（nav 结构不变，类名不变）

### 搜索增强

**首页搜索：** 保持现有逻辑（过滤 `.module-card`）

**模块列表页搜索：** 新增按页面类型过滤
- 首页: `.module-card` 文本匹配
- 列表页（`.list-page`）: `.list-item` 文本匹配
- 详情页: 搜索触发全局模块跳转（输入关键词 → 下拉提示匹配的模块/页面）

**实现：** 扩展 `js/main.js` 中的搜索逻辑，增加场景判断。

## 二、索引页设计统一

**问题：** 11 个模块索引页风格不一致——快反/反恐已有分组列表，警情/装备/巡逻等使用扁平列表，作战单元 9 篇完全平铺。

**方案：** 全模块统一为分组列表设计，与快反/反恐保持一致。

**格式规范：**
```html
<div class="list-section-title">📌 分组标题</div>
<a href="page.html" class="list-item">
  <div class="item-title">emoji 页面标题</div>
  <div class="item-desc">简短描述 · 关键词列表</div>
</a>
```

**各模块分组计划：**

| 模块 | 分组 | 文章数 |
|------|------|--------|
| **警情** | 📌 街面警情 / ⚠️ 重点警情 | 3+3=6 |
| **装备** | 📦 核心装备 / 📢 规范用语 | 6+1=7 |
| **巡逻** | 📌 勤务规范 / 🔍 专项盘查 | 3+2=5 |
| **作战单元** | 📐 基础认知 / ⚔️ 装备技术 / 📊 战术训练 | 3+3+3=9 |
| **徒手防卫** | 📌 理论基础 / 🥋 实战技术 | 2+2=4 |
| **手枪射击** | 📌 理论认知 / 🎯 射击技术 / 🔧 维护保养 | 2+2+1=5 |
| **大型活动安保** | 📌 场景执法 / 🏟️ 专项保障 | 2+2=4 |
| **法律法规** | 📌 执法规范 / ⚖️ 专项法规 | 2+2=4 |
| **群体事件** | 📌 理论机制 / ⚔️ 实战处置 | 2+3=5 |

> 快反、反恐已采用分组列表，无需修改。

**不改:** 所有详情页内容、CSS 样式、面包屑、页面元信息。

## 三、首页信息密度优化

**问题：** 首页 4 列卡片网格信息量少，用户需逐个点进模块才知道里面有什么。

**方案：** 在现有卡片内增加文章数和热门链接，不改变 4 列网格布局。

**卡片结构：**
```html
<a href="zhuangbei/index.html" class="module-card accent-blue">
  <div class="icon">🛡️</div>
  <div class="name">单警装备</div>
  <div class="count">7篇</div>
  <div class="desc">九小件使用规范</div>
  <div class="hot-links">
    伸缩警棍 · 催泪喷射器 · 手铐…
  </div>
</a>
```

**新增 CSS：**
- `.module-card .count` — 小字灰色，文章数量
- `.module-card .hot-links` — 更小字，用 `·` 分隔，3-4 个热门子页面

**卡片比现在高约 20px，响应式断点不变。**
