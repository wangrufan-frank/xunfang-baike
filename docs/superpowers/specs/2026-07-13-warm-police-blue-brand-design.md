# 巡防百科「暖警蓝」品牌升级设计

## 目标

在不改变巡防百科现有首页结构、栏目分类、搜索逻辑和内容链接的前提下，建立更有温度的基层服务视觉风格。网站应同时传达警务工具的可靠性与一线巡防人员可随时使用的亲和感。

目标用户为一线巡防民警和辅警，主要在手机巡逻间隙快速查阅内容。

## 范围

本次升级覆盖全站共享视觉系统及其在首页、栏目列表、文章页、搜索页和“每月一学”页面的表现：

- 全局色彩、文字层级、圆角、边框、阴影和交互状态。
- 顶部导航、搜索框、模块卡片、步骤卡、标签、按钮和搜索结果卡。
- 桌面端与手机窄屏的间距、触控尺寸和视觉层级。

不包含新功能、账户体系、收藏、后台、内容重写或大规模新增图片素材。

## 品牌方向：暖警蓝

采用低饱和警务蓝作为专业骨架，以暖白和雾蓝建立舒适阅读环境，以琥珀橙提供克制的服务感与重点提示。

| 用途 | 色值 | 说明 |
| --- | --- | --- |
| 主警蓝 | `#1F4E79` | 顶部导航、关键标题、主要操作 |
| 深警蓝 | `#163A5C` | 导航悬停、强调文字、深色区域 |
| 雾蓝 | `#EAF2F8` | 页面辅助背景、轻提示、标签底色 |
| 暖白 | `#FFFCF7` | 主页面背景与卡片内的阅读基调 |
| 琥珀橙 | `#D98B2B` | 当前栏目、重点提示、“每月一学”亮点 |
| 正文深灰蓝 | `#243447` | 正文与重要信息 |
| 次级文字 | `#60758C` | 说明、元数据和辅助信息 |
| 边框 | `#D8E2EC` | 卡片与输入框边界 |

## 组件规则

### 导航与搜索

- 顶部导航使用主警蓝；当前栏目使用琥珀橙色文字或细线标识，不使用大面积高饱和红色。
- 保留现有手机折叠菜单行为。链接、路径计算与菜单结构不变。
- 搜索框使用暖白底、12px 圆角和更清晰的内边距；聚焦时使用雾蓝外圈或主警蓝边框。

### 卡片与内容块

- 全站卡片使用暖白背景、12px 圆角、浅色边框和轻阴影；悬停仅做小幅上移或边框加深。
- 首页六个分类卡片继续保留，不改变排序、数量、链接或文案。去除彩虹式强色顶部边框，改为统一的警蓝图标与低饱和分类提示。
- 文章步骤卡和内容列表卡遵循相同圆角、边框和文字层级；风险和重要内容可继续使用语义色，但降低色块面积。
- 标签和元信息使用雾蓝底或暖白描边；不承担主要操作功能。

### 每月一学

- 保留现有“每月一学”的识别作用。
- 使用深警蓝到主警蓝的背景，搭配琥珀橙操作入口与暖白文字，成为全站稳定的特色亮点。

## 页面表现

- 首页保留现有搜索、每月专题和六模块卡片结构；只更新色彩、留白、圆角与视觉层级。
- 栏目列表、文章页和搜索页通过统一卡片与文字规范获得连续体验，不更改信息架构。
- 手机窄屏保持两列模块卡片和折叠导航；所有可点击控件保持足够触控空间，正文和说明文字具备清晰对比度。

## 可访问性与降级

- 正文、次级文字、按钮和导航状态与各自背景保持可读对比度。
- 色彩不作为唯一的信息传达方式；风险级别继续使用文字、标签或图标辅助。
- 新样式仅覆盖共享 CSS。浏览器不支持阴影或圆角时，页面仍以边框和色彩保持清晰结构。

## 验收标准

- 首页、至少一个栏目列表页、至少一个文章页、搜索页及“每月一学”页面均使用统一的暖警蓝视觉语言。
- 现有导航、搜索、每月一学入口和相对链接保持可用。
- 在桌面和手机窄屏下检查导航展开、搜索下拉、卡片间距、文字对比度及控制台错误。
- 不修改 `search-index.json`，不修改内容数据和页面链接。
