# 每月一学模块设计文档

**日期:** 2026-07-08
**状态:** 待实施

---

## 概述

将首页嵌入式"每日一学"改造为独立"每月一学"品牌模块。每月一期深度长文，首页大幅精选横幅 + 杂志化内页，内容源自 Obsidian 警务技能知识库。

## 定位

区别于六大模块的知识分类定位，每月一学是网站的**品牌模块** — 定期更新的深度内容，首页第一视觉重心。

## 架构

### 目录结构

```
/meiyueyixue/
  index.html          ← 当月内容（完整长文）
  data.js             ← 元数据配置（主题、日期、摘要、文件映射）
  YYYY-MM.html        ← 历史月份归档页面
```

### 首页入口

- 搜索栏与 6 模块卡片之间插入大幅精选横幅
- 横幅占满内容宽度，深色底 + 金色点缀，与手册暖色调形成反差
- 横幅内容：月份标签 + 主题标题 + 摘要 + 「查看全文 →」按钮
- JS 读取 `data.js` 动态渲染，切换月份自动更新

### 导航栏入口

- `nav.js` MODULES 数组新增第 7 项「本月精选」
- 渲染为金色高亮按钮，样式区别于其他导航项
- 可选：导航栏右侧独立展示，与其他模块左对齐项形成视觉分离

### 内页布局（杂志风格）

```
大标题区：
  月份副标题（如 2026年7月）
  主题主标题
  简短引语 / 本期导读

封面图区：
  当月主题配图（可选，无图时纯文字布局也成立）

正文区（五段式，可灵活调整）：
  一、场景概述
  二、核心要点
  三、经典案例复盘
  四、法规依据速查
  五、实操注意事项

参考资料区：
  Obsidian 笔记链接列表

往期精选区：
  卡片网格，时间倒序，每张卡片显示月份 + 主题
```

- 单页滚动，不分页
- 移动端适配：封面图缩至全宽，五段式保持单列阅读

## 内容工作流

### 素材积累（日常）

用户在 Obsidian `03_Knowledge/警务技能/` 目录下持续积累笔记，覆盖巡逻盘查、反恐防暴、徒手防卫、执法规范等分类。素材形式不限：案例记录、法规摘录、培训心得、教程笔记。

### 每期制作流程

1. **扫描提案** — Claude 扫描知识库近期新增/更新笔记，找出素材丰富、适合深挖的方向，提案 2-3 个主题
2. **用户选定** — 用户确认当月主题
3. **精读整合** — Claude 精读相关笔记，结合外部前沿警务规范，生成五段式长文初稿
4. **审阅定稿** — 用户审阅，提出修改意见，Claude 修改后定稿
5. **上线发布** — 生成 HTML 页面，更新 `data.js`，更新首页横幅，提交部署

### 两种内容模式

- **合成模式**：多源笔记整合 + 外部规范补充 → 结构化长文
- **改编模式**：单篇优质笔记直接改编为当月长文

## 配置驱动

`data.js` 结构：

```js
var monthlyData = {
  current: "2026-07",
  articles: {
    "2026-07": {
      theme: "主题标题",
      summary: "简短摘要（首页横幅使用）",
      author: "巡防百科编辑部",
      tags: ["标签1", "标签2"],
      image: "img/monthly/2026-07-cover.jpg",  // 可选
      file: "2026-07.html"
    }
  }
};
```

- `current` 指向当月，首页横幅读取此字段
- `articles` 按 YYYY-MM 键值存储，往期列表按日期倒序自动生成
- 新增月份只需：一篇 HTML + 更新 `data.js`（一条记录）+ 可选配图

## 视觉规范

### 首页横幅

| 属性 | 值 |
|------|-----|
| 底色 | 深色 `#1a1a2e` 或 `#2c3e50`（导航栏同系更深） |
| 标题色 | 金色 `#d4a843` |
| 正文字色 | 浅色 `#ecf0f1` |
| 圆角 | 8px |
| 阴影 | 0 4px 12px rgba(0,0,0,0.15) |

### 内页杂志

- 大标题字号 2rem+，加粗，深棕 `#3e2723`
- 正文 1rem，行高 1.8，最大宽度 800px 居中
- 五段式每段有小标题（带金色左侧边线）
- 往期卡片与现有模块卡片风格一致，缩略展示

## 错误处理

- 封面图加载失败：退化为纯文字布局，不影响阅读
- `data.js` 加载失败：横幅区域不渲染（不阻塞首页其他内容）
- 往期页面不存在：索引页显示「即将上线」占位
