# 巡防百科教学视频自然旁白优化设计

## 目标

将现有教学视频中的机械化女声替换为更接近内部培训讲解的自然中文女声；不修改网页截图、字幕、镜头顺序或视频视觉设计。

## 方案选择

采用联网神经语音合成，使用普通话女声、约 0.92 倍自然语速。与本地 Windows TTS 相比，神经语音在连读、停顿和语气收束方面更自然；代价是首次生成需要联网。

## 文案与节奏

沿用六段讲解的事实内容，微调为更口语化的播报稿：

- 按句意使用逗号、顿号和省略停顿，避免逐字平铺。
- 在“统一入口”“六大业务模块”“站内搜索”“核心结论”等关键词前后留出短停顿。
- 网址播报为“巡防百科点 C N”，避免字母连读不清。
- 保持总时长接近原 2 分 15 秒；若新旁白略长或略短，仅调整各场景持续时间和字幕出现时机，不改变画面内容。

## 产物与验证

生成新的 WAV 音频、更新分段时间表，并以现有 1080P H.264 视频画面重新封装为 MP4。验收标准：文件包含 H.264 视频和 AAC 音轨、时长与旁白相符、开头/结尾无静音截断、在 Windows 默认播放器中可直接播放。

## 范围边界

不改变网站源码、PPT、画面布局或原始讲解主题；不上传任何网站登录信息、个人资料或敏感内容。
