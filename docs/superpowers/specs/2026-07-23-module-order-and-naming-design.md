# 网站模块顺序与命名优化设计

## 目标

统一调整巡防百科正式网站六大模块的展示顺序和用户可见名称，使首页、全站导航、模块页面、详情页面与搜索内容保持一致，同时维持现有 URL 的稳定性。

## 模块顺序与名称

正式网站的六大模块统一按以下顺序展示：

1. 警情处置
2. 勤务须知（原“勤务保障”）
3. 执法规范
4. 装备操作（原“装备介绍”）
5. 教育学习（原“教育培训”）
6. 实战训练（原“警务训练”）

“本月精选”等不属于六大模块的入口继续位于六大模块之后，并保持现有视觉样式。

## 改动范围

### 首页与全站导航

- 调整首页六张模块卡片的 DOM 顺序和显示名称。
- 调整全站顶部导航中六个模块入口的顺序和显示名称。
- 保持各模块现有图标、颜色、简介、篇目数量和链接不变。

### 正式网站内容

在正式网站运行内容中统一替换旧模块名称，包括：

- 页面 `<title>`；
- 模块首页标题；
- 面包屑；
- 返回模块入口；
- 正文中的模块交叉引用；
- 搜索索引中的用户可见标题、分类或描述；
- 内容清单中用于网站运行或维护的模块显示名称。

未发生改名的“警情处置”和“执法规范”只调整展示顺序，不修改其名称。

### 路径兼容

保留现有目录和链接：

- `jingqing/`
- `qinwu/`
- `fagui/`
- `zhuangbei/`
- `zoufang/`
- `xunlian/`

本次不迁移目录、不修改 URL，也不增加重定向。

## 不在范围内

- 历史设计文档和实施计划；
- 汇报文档、演示文件、截图和其他交付物；
- 微信小程序；
- 与模块顺序或名称无关的页面布局、样式和内容重构。

## 验证

新增或更新自动化检查，验证：

1. 首页六张模块卡片按目标顺序出现；
2. 全站导航六个模块按目标顺序出现；
3. 正式网站用户可见内容不再出现“勤务保障”“装备介绍”“教育培训”“警务训练”四个旧名称；
4. 六个模块链接继续指向原有路径；
5. 现有网站结构、站内链接和搜索相关测试继续通过。
