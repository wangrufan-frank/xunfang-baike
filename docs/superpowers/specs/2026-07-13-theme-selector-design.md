# 巡防百科主题选择设计

## 目标

让学习者手动选择网站外观，并在后续访问时保留选择。默认保持现有暖警蓝；升级前的暖棕视觉作为“经典暖棕”主题保留。

## 主题

- `warm-police-blue`：默认暖警蓝。
- `classic-warm`：升级前的暖棕配色。
- `daylight`：高可读性日间浅色。
- `night`：低亮度夜间深色。

主题通过 `html[data-theme]` 覆盖共享 CSS 变量，不复制页面或组件样式。

## 交互与存储

- 桌面端在顶部导航右侧提供“外观”下拉入口；手机端入口位于折叠菜单。
- 选择主题后立即应用，无页面刷新，并以当前主题名称显示选中状态。
- `js/theme.js` 使用 `localStorage` 键 `xunfang-theme` 保存选择；加载时优先应用已保存主题，否则应用 `warm-police-blue`。
- 无法读写 `localStorage` 时静默使用默认主题，不影响导航、搜索或内容访问。

## 实现边界

- 新增 `js/theme.js`，负责验证、读取、应用和保存主题。
- `js/nav.js` 只增加主题选择控件及事件绑定，保留现有模块链接、路径计算和移动菜单行为。
- 全部页面在 `nav.js` 前加载 `theme.js`，避免可见的首屏主题闪烁。
- 不修改内容、链接、搜索索引或数据文件。

## 验收

- 四种主题均能切换，且导航、搜索、卡片、步骤、月度专题文字保持可读。
- 刷新页面和进入不同栏目后，选择的主题仍然存在。
- 在桌面及手机折叠菜单中均能触发主题选择。
- 默认主题为暖警蓝；本地存储不可用时页面仍可正常使用。
