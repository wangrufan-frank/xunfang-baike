# 网站密码保护 — 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为巡防百科网站添加客户端JS密码门，密码150225，SHA-256哈希，cookie有效期7天。

**Architecture:** `js/main.js` 顶部加同步认证守卫（页面渲染前执行），未认证跳转 `auth.html` 登录页。`auth.html` 内部跳过守卫逻辑。

**Tech Stack:** 纯 HTML/CSS/JS，SHA-256（SubtleCrypto API）

## Global Constraints

- 密码：150225，SHA-256 散列存储，不存明文
- Cookie 名：`xunfang_auth`，值用 SHA-256(password) 结果
- Cookie 有效期：7 天
- `auth.html` 自身不触发认证守卫
- 登录成功后跳转到 `redirect` URL 参数指向的页面（无则跳首页）
- 不改动其他 HTML 文件、CSS、nav.js
- 响应式适配移动端

(async function() {
  // 150225 -> SHA-256 hex
  var VALID = '5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8';
  var COOKIE = 'xunfang_auth';

  function getParam(name) {
    var m = location.search.match(new RegExp('[?&]' + name + '=([^&]*)'));
    return m ? decodeURIComponent(m[1]) : '';
  }

  async function sha256(text) {
    var buf = new TextEncoder().encode(text);
    var hash = await crypto.subtle.digest('SHA-256', buf);
    return Array.from(new Uint8Array(hash)).map(function(b) { return b.toString(16).padStart(2,'0'); }).join('');
  }

    var pwd = document.getElementById('pwd').value.trim();
    if (!pwd) { document.getElementById('error').textContent = '请输入密码'; return; }
    var h = await sha256(pwd);
    if (h !== VALID) {
      document.getElementById('error').textContent = '密码错误';
      document.getElementById('pwd').value = '';
      return;
    }
    var d = new Date();
    d.setDate(d.getDate() + 7);
    document.cookie = COOKIE + '=' + h + '; expires=' + d.toUTCString() + '; path=/; SameSite=Lax';
    var redir = getParam('redirect');
    location.href = redir || 'index.html';
  });

  document.getElementById('pwd').addEventListener('keydown', function(e) {
    if (e.key === 'Enter') document.getElementById('submit').click();
  });
})();
</script>
</body>
</html>

### Task 2: 修改 js/main.js — 添加认证守卫

**Files:**
- Modify: `js/main.js`

- [ ] **Step 1: 在 main.js 顶部插入认证守卫代码**

在 `js/main.js` 最开头（第1行之前）插入以下代码。这段代码同步执行，在 DOM 解析前就完成检查，未认证直接跳转不渲染页面：

```javascript
// ── 认证守卫 ──
(function(){
  var COOKIE = 'xunfang_auth';
  // 150225 -> SHA-256
  var VALID = '5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8';
  var isAuthPage = location.pathname.endsWith('/auth.html') || location.pathname === '/auth.html';
  if (isAuthPage) return;
  var val = document.cookie.split('; ').reduce(function(acc, c) {
    var kv = c.split('='); acc[kv[0]] = kv.slice(1).join('='); return acc;
  }, {})[COOKIE];
  if (val !== VALID) {
    var redir = encodeURIComponent(location.pathname.replace(/^.*[\\\/]/, '') + location.search);
    location.replace('auth.html?redirect=' + redir);
  }
})();
```

- [ ] **Step 2: 验证**

1. 清除所有 cookie，打开 `index.html` → 应自动跳转 `auth.html`
2. 从 `auth.html` 输入密码 → 应成功进入 `index.html`
3. 刷新 `index.html` → 不应再跳转（cookie 已存在）
4. 导航到详情页（如 `zhuangbei/shensuo-jinggun.html`）→ 不再跳转
5. 打开 F12 → Application → Cookies，确认 `xunfang_auth` 存在，有效期7天
